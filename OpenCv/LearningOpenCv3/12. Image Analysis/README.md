# 离散傅里叶变换
### 离散傅里叶变换原理
说明：对一张图像使用傅里叶变换就是将它分解成正弦和余弦两部分，也就是将图像从空间域(spatial domain)转换到频域(frequency domain)。傅里叶变换就是一个用来将函数分解的工具，任一函数都可以表示成无数个正弦和余弦函数的和的形式。
